Real terrain maker is an easy tool for creating terrain with the real geographic data.
Specifies the geographic coordinates, and make a terrain of anywhere on the earth in a few clicks.
This plug-in runs depending on the Google/Bing/mapbox map service, which requires Elevations API and Static map API keys.
Features:

◆Real Elevation data:

Using the elevation data of Google map, Bing map, and Mapbox, over 30 meters in most of the Earth areas. When you meet problems such as lack of data in a certain area, try another data source if one is not so satisfactory.

◆Easy setting:

To make a terrain faster or in high quality, simulate the real world or make an extravagant land. All setting can be made in a few clicks. Your keys and settings will be saved, no need to remember the location you generated last night.

◆Save in many types:

The data downloaded can be saved in mesh, terrain, and prefab. The material is created automaticity. You can share the terrain you made with others without a hitch. With a combine and save tool, You can combine and save the generated mesh to prefab and unity terrain at any time.

◆Basic Materials for Terrain Provided:

Several materials such as contour, grid, gradient and multiple texture height shader. It's possible to add some snow at the top of mountains and lava in the valley.

◆ Height Map:

A heightmap file will be saved after the terrain generated(optional). You can generate a new terrain with the premade height map at any time.

◆Style Setting:

You can generate a terrain of low-poly style and set a default material before generating them. You can also set the default layer for the use of Infinite Map.

◆ Map Style:

Besides satellite, many other styles of terrain texture are supported. It is convenient for making a 3D map. Image saved in a folder for each time the generate starts(optional).

◆ Preset template:

Include some common options for fast data preview or high details terrain creating.

◆ Random Map:

One selectable option to generate a random map is provided. It can be used to save time when doing tests or just make an infinite random terrain in a game.

◆Run Time Infinite Map:

Download the map texture and elevation data automaticity while running the game. The adjacent tiles will be generated while the character moving.

◆ Diagnosis Tool:

Check the network condition and API key state. If you are not sure about why RTM work incorrectly, try the Diagnosis tool to find out.

◆ Smooth style:

Make a smooth terrain with a smaller sample.
