slideshow Effects Guide Script Home
Dynamic materials for making animations on your slide of slideshow effects.
Also include AnimationEffectHelper and SlideshowHelper to make a slideshow with many pictures.

SlideShowHelper/HalftoneHelper:

Example of making slideshows with Wheel/Halftone Effects/Triangular Billboard.

Animation Effect Helper:

A simple tool of making animations based on time.
Add a Keyword element to the Key List, fill out the Time and Value of each waypoint.
You can also use the Keyword Assistant to add some frequently used keywords.

Keywords of each material:
